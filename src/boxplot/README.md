### usage
```bash
Rscript boxplot.R -h

Usage: boxplot.R [options]

Options:
	-i INFILE, --infile=INFILE
		The input file.

	-o OUTFILE, --outfile=OUTFILE
		The output file.

	--title=TITLE
		The main title[default "Boxplot"].

	--xlab=XLAB
		The x axis label[default "X label"].

	--ylab=YLAB
		The y axis label[default "Y label"].

	--width=WIDTH
		The width of the picture.

	--height=HEIGHT
		The height of the picture.

	-h, --help
		Show this help message and exit

	Contact: suqingdong@novogene.com
```
