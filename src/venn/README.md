### Examples
```bash
Rscript venn.R -h

Usage: Rscript venn.R [Options]

Options:
	-i INFILE, --infile=INFILE
		The input file.

	-o OUTFILE, --outfile=OUTFILE
		The output file.

	--height=HEIGHT
		The height of image[default=1000].

	--width=WIDTH
		The width of image[default=1000].

	--main=MAIN
		The main title of image[default=""].

	-h, --help
		Show this help message and exit

```
